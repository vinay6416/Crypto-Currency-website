# 💰 Crypto Currency Website

A modern and responsive Crypto Currency Website built using **HTML, CSS, and JavaScript**.

This project displays cryptocurrency cards, live price updates, smooth scrolling navigation, animated UI effects, and a modern gradient design.

---

# 🚀 Features

- Working Navigation Bar Links
- Smooth Scrolling Functionality
- Get Started Button Navigation
- Live Cryptocurrency Price Updates
- Crypto Cards Section
- About Section
- Contact Section
- Animated Hero Image
- Sticky Navigation Bar
- Hover Animations
- Responsive Design
- Modern Gradient Background UI

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
